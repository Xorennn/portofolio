Basic HTML Tabel

Dalam dokumen HTML, tabel dapat dibuat menggunakan elemen <table>. Tabel terdiri dari baris (<tr>) dan kolom (<td> untuk data dan <th> untuk header). Berikut adalah langkah-langkah untuk membuat tabel:

  1. Gunakan elemen <table> untuk memulai tabel.
  2. Tambahkan elemen <thead> untuk bagian header tabel, dan di dalamnya gunakan <tr> untuk baris dan <th> untuk kolom header.
  3. Gunakan elemen <tbody> untuk bagian isi tabel, dan di dalamnya tambahkan baris menggunakan <tr> dan kolom data menggunakan <td>.
  4. Anda dapat menambahkan gaya menggunakan CSS untuk mempercantik tabel.
